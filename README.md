Build
To build and run tests, do the following

npm ci

npm run build
Test
npm run test
Gas Report
Generate gas report

REPORT_GAS=true npm run test
Coverage
Generate test coverage report

npm run coverage
Test coverage reports of the main branch can also be found on Codecov.
